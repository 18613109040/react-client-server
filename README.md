# 前编码规范
### 文件名命名规则
###### 样式文件命名规则
```
使用驼峰： 如searchBound.scss
```
###### JS&JSX文件命名规则
```
使用驼峰： 如SearchBound.scss
```
###### 缩进
```
对于所有编程语言，缩进必须是软tab（用空格字符）。在你的文本编辑器里敲 Tab 应该等于 2个空格 。
```
###### 变量使用驼峰或全小写
```
let varName
```
##### 样式里的class有多个单词的用“-”链接
```
<div class="class-name"></div>
```
##### 样式里的ID使用者驼峰
```
<div id="idName" class="class-name"></div>
```
